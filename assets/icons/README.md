# Social Login Icons

Add your social login icons here:

## Required Icons:

1. **google.png** - Google "G" logo (24x24 or 48x48 recommended)
2. **facebook.png** - Facebook "f" logo (24x24 or 48x48 recommended)

## Where to get icons:

- Google Brand Guidelines: https://developers.google.com/identity/branding-guidelines
- Facebook Brand Resources: https://en.facebookbrand.com/

## Format:

- PNG with transparent background
- Size: 48x48px or 24x24px for standard buttons
- Use official brand colors

## Alternative:

If you don't have image assets, the code will fall back to Material Icons:
- `Icons.g_mobiledata` for Google
- `Icons.facebook` for Facebook
